# About the Mburicaó Monitoring Project

<div class="page-content">
    <div class="container">
        <h2>Urban Hydrological Monitoring & Citizen Science</h2>
        
        <div class="info-box">
            <h3><i class="fas fa-project-diagram"></i> Project Overview</h3>
            <p>The Mburicaó Monitoring Project is a comprehensive urban hydrological monitoring network in Asunción, Paraguay. It combines IoT sensor technology, machine learning models, and citizen science to provide early warning systems for flooding events in rapid urban streams.</p>
            <p>Our mission is to support sustainable water resource management and community resilience in the face of increasingly extreme weather events.</p>
        </div>

        <div class="info-box">
            <h3><i class="fas fa-chart-line"></i> Data & Technology</h3>
            <ul style="margin-left: 1.5rem;">
                <li><strong>Monitoring Frequency:</strong> 10-minute intervals for all sensors</li>
                <li><strong>Sensor Types:</strong> Water level sensors, precipitation gauges, meteorological stations</li>
                <li><strong>Prediction Models:</strong> Support Vector Machine (SVM) and Multilinear Regression</li>
                <li><strong>Data Visualization:</strong> Real-time monitoring dashboards with Grafana</li>
                <li><strong>Geographic Focus:</strong> Mburicaó Stream, Asunción, Paraguay</li>
            </ul>
        </div>
